## Create shortcut from the Start menu

First open the desired folder for the shortcut.

*(Click on the individual step or triangle to hide or show the details (images, info, ...))*

{{< collapse summary="**Step 1:** Left mouse button click on the `Start` button" openByDefault=true >}}

   {{< figure src="/images/other/win10/Win10en_-_start_btn_start.jpeg" title="In Windows 10" >}}
   {{< figure src="/images/other/win11/Win11en_-_start_btn_start.jpeg" title="In Windows 11" >}}

{{< /collapse >}}

{{< collapse summary="**Step 2:** Find the desired element for the shortcut" openByDefault=true >}}

   Inside `Windows 11` `Start menu` looks different than in `Windows 10` so if you want to see the list of the programs, left mouse button click on the `All >` button.

   {{< figure src="/images/other/win11/Win11en_-_start_btn_-_all_btn.jpeg" >}}

{{< /collapse >}}

{{< collapse summary="**Step 2a** Drag and drop method" openByDefault=true >}}
   
   Click and hold left mouse button on the the desired element, drag it to the desired location (Start menu pinned apps, taskbar, desktop or to some folder) and then release mouse button. Now shortcut is created.

   {{< figure src="/images/other/win10/Win10en_-_start_btn_-_excel_drag.jpeg" title="In Windows 10" >}}
   {{< figure src="/images/other/win11/Win11en_-_start_btn_-_adobe_acrobat_drag.jpeg" title="In Windows 11" >}}

{{< /collapse >}}

{{< collapse summary="**Step 2b** Right mouse button menu" openByDefault=true >}}

   Right mouse button click on the desired element and then:
   
   - Left mouse button click on the `Pin to Start` to pin the shortcut in the Start menu.

   {{< figure src="/images/other/win10/Win10en_-_start_btn_-_excel_rmb_-_pin_start.jpeg" title="In Windows 10" >}}
   {{< figure src="/images/other/win11/Win11en_-_start_btn_-_adobe_acrobat_rmb_-_pin_start.jpeg" title="In Windows 11" >}}

   - With mouse pointer go over `More` and then left mouse button click on the `Pin to taskbar` to add shortcut in the Windows taskbar.

   {{< figure src="/images/other/win10/Win10en_-_start_btn_-_excel_rmb_-_more_-_pin_taskbar.jpeg" title="In Windows 10" >}}
   {{< figure src="/images/other/win11/Win11en_-_start_btn_-_adobe_acrobat_rmb_-_more_-_pin_taskbar.jpeg" title="In Windows 11" >}}

{{< /collapse >}}

## Shrotcuts from the Windows File Manager

First open the desired folder for the shortcut then the location of the element you want to make shortcut of. You can also use only one File manager window if you want.

*(Click on the individual step or triangle to hide or show the details (images, info, ...))*

{{< collapse summary="**Drag and drop** method" openByDefault=true >}}

   With left mouse button click and hold the desired element and drag it to the desired location. Before you release mouse button check that you have `Create link` written othervise keyboard click and hold left `Alt` button on the keyboard, first release mouse button and then keyboard button. Now shortcut is created.
   
   *Keyboard button options when dragging:*

   {{< figure src="/images/other/win10/Win10en_-_file_manager_-_chrome_drag_shortcut_alt_key.jpeg" title="Left Alt for shortcut" >}}
   {{< figure src="/images/other/win10/Win10en_-_file_manager_-_chrome_proxy_drag_shortcut_ctrl_key.jpeg" title="CTRL to copy" >}}
   {{< figure src="/images/other/win10/Win10en_-_file_manager_-_chrome_proxy_drag_shortcut_shift_key.jpeg" title="SHIFT to move" >}}

{{< /collapse >}}

{{< collapse summary="**Right mouse button menu** method" openByDefault=true >}}

   - Right mouse button click on the desired element, go over the `Send to` and then left mouse button click on the `Desktop (create shortcut)` or what ever option you want.
   
   {{< figure src="/images/other/win10/Win10en_-_file_manager_-_chrome_rmb_-_send_to_-_desktop_create_btn.jpeg" >}}

   - Right mouse button click on the desired element and then left mouse button click on the `Create shortcut`. If folder is protected you will see an error window and with left mouse button click `Yes` button if you want to create shortcut on the desktop instead but if you want to cancel the progress, left mouse button click on `No`.
   
   {{< figure src="/images/other/win10/Win10en_-_file_manager_-_chrome_rmb_-_create_shortcut_btn.jpeg" >}}
   {{< figure src="/images/other/win10/Win10en_-_file_manager_-_chrome_rmb_-_create_shortcut_btn_-_error_yes.jpeg" title="Protected folder error" >}}

   **To see the `Send to` and `Create shortcut` in `Windows 11` first you need to left mouse button click on the `Show more options` button.**

   {{< figure src="/images/other/win11/Win11en_-_chrome_rmb_-_show_more_options_btn.jpeg" >}}

{{< /collapse >}}

## Manually create the shortcut

*(Click on the individual step or triangle to hide or show the details (images, info, ...))*

{{< collapse summary="**Step 1:** Open the desired folder for the shortcut" openByDefault=true >}}

{{< /collapse >}}

{{< collapse summary="**Step 2:** Right mouse button click in the empty area of the desired folder, go over `New` and then left mouse button click on the `Shortcut` button" openByDefault=true >}}

   {{< figure src="/images/other/win10/Win10en_-_desktop_rmb_-_new_-_shortcut_btn.jpeg" title="In Windows 10" >}}
   {{< figure src="/images/other/win11/Win11en_-_desktop_rmb_-_new_-_shortcut_btn.jpeg" title="In Windows 11" >}}

   Inside the the new window if you want to copy and paste or write the location, continue reading this step to write or copy and paste the location but if you want manually select the element skip to `Step 3`.
   
   - **Website address** needs to contain `www.` or refference like `http://`, `https://`, `ftp://`, ... without at least one of them for example `aconetwork.net` Windows does not regnonize it as a website address. Good examples: `https://aconetwork.net` OR `www.aconetwork.net`
      
   - **File or folder location** example: `C:\Program Files\Google\Chrome\Application\chrome.exe`

   Now skip to the `Step 5`

{{< /collapse >}}

{{< collapse summary="**Step 3:** With left mouse button click the `Browse` button" openByDefault=true >}}

   {{< figure src="/images/other/win10/Win10en_-_create_shortcut_-_location_-_browse_btn.jpeg" >}}

{{< /collapse >}}

{{< collapse summary="**Step 4:** Navigate to a desired element location, left mouse button click on the element and then left mouse button click on the `OK` button" openByDefault=true >}}

   To expand folders with left mouse button click on the arrows located on the left side of the window until you get to the desired location.
   
   {{< figure src="/images/other/win10/Win10en_-_create_shortcut_-_browse_-_arrow_btn.jpeg" title="Arrows" >}}

   Now left mouse button click on the desired element, example:

   {{< figure src="/images/other/win10/Win10en_-_create_shortcut_-_browse_-_chrome.jpeg" >}}

   To confirm selected element left mouse button click on the `OK` button and this will automatically populate text box in the previous window.

   {{< figure src="/images/other/win10/Win10en_-_create_shortcut_-_browse_-_ok_btn.jpeg" >}}

{{< /collapse >}}

{{< collapse summary="**Step 5:** Left mouse button click on the `Next` button" openByDefault=true >}}
   
   {{< figure src="/images/other/win10/Win10en_-_create_shortcut_-_next_btn.jpeg" >}}

{{< /collapse >}}

{{< collapse summary="**Step 6:** In the new text box type in the desired name and then left mouse button click on the `Finish` button to finish creating the shortcut" openByDefault=true >}}

   Text box is under the `Type the location of the item:`.

   {{< figure src="/images/other/win10/Win10en_-_create_shortcut_-_finish_btn.jpeg" >}}
